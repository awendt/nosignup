Websites that require no signup

There are tons of websites out there requiring you to create a user account and login, but few of them really need to do that. This overview promotes websites that can be used without signing up. Click on a category to view it.

Event planning

hyggelig.org
Simple event planning for you and your friends
Create an event page and send its URL to your guests. They accept or decline online. Provides custom URL, multilingual interface, RSS feed, and source code.
Mobaganda*
Propaganda for your friends
Again, create an event page and send your guests the URL. Provides a custom URL, global email address for all guests and RSS feed, but closed source and English-only interface.

File sharing

Drop.io
The simplest way to share online
Drop.io lets you share files by web, e-mail, phone. Create a drop and start sharing. Drops are protected from search engines. Requires Flash 9.

Reminders

Oh don't forget...
SMS text message reminder app, with ease.
Send reminders to a specific cell number to get reminded about anything.
